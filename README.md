This is a Top-Down Game like interactive map for Thapathali Campus, IOE.

You can control the character using WSAD keys. Q key can be use to interact.

E-block is accessible. Press 'q' on the entry(stairs) of  Eblock. You can go to every floor of E-block.
Canteen is also accessible.
Other Blocks are not accesible.
